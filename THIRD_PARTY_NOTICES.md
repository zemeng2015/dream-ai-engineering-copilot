# Third-Party Notices

DREAM source code is licensed under Apache-2.0. The project depends on
third-party packages installed from standard package registries at development
or runtime. This file records the current dependency license posture for
company-import review; it is not a substitute for a formal SBOM generated by a
company compliance scanner.

## Python Runtime and Development Dependencies

Declared Python dependencies are listed in `pyproject.toml`.

- FastAPI: MIT
- Pydantic: MIT
- PyYAML: MIT
- Typer: MIT
- Uvicorn: BSD-3-Clause
- HTTPX: BSD-3-Clause
- pytest: MIT
- Ruff: MIT

Transitive dependencies should be reviewed from the installed environment or a
generated lock/SBOM before production redistribution.

## Frontend Dependencies

Frontend dependencies are listed in `frontend/package.json` and locked in
`frontend/package-lock.json`.

The current lockfile is dominated by permissive licenses, including MIT, ISC,
Apache-2.0, BSD-2-Clause, BSD-3-Clause, 0BSD, BlueOak-1.0.0, CC0-1.0,
CC-BY-3.0, CC-BY-4.0, and Python-2.0.

One transitive development dependency, `node-forge`, is declared as
`(BSD-3-Clause OR GPL-2.0)`. DREAM elects to use this dependency under the
BSD-3-Clause option.

## Vendored Code

This repository does not intentionally vendor third-party source packages.
Generated install folders such as `.venv/`, `node_modules/`, build outputs, and
local artifacts must remain untracked.
